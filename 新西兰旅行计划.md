# 新西兰南岛冬季自驾旅行计划
v1.2
**行程日期：7 月 11 日 – 7 月 17 日**  
**旅行方式：自驾 + Milford Sound 一日巴士团**  
**核心路线：Christchurch → Twizel → Aoraki / Mt Cook → Wanaka → Queenstown → Twizel → Christchurch**

本计划为 **7.11–7.17 南岛自驾** 的详细行程。核心安排是：**7.13 早上 Mt Cook，7.14 下午 Queenstown 跳伞，7.15 Milford Sound 巴士团，7.16 回 Twizel 做返程缓冲，7.17 回 Christchurch 返程**。

7 月是冬季，下面所有驾驶时间都按正常路况估算；实际请额外预留 **20–30%**，尤其是 Lindis Pass、Crown Range、Mt Cook Road、Milford Road 这类山路。

---

## 行程总览

| 日期   | 路线                                                                      | 住宿           |
| ---- | ----------------------------------------------------------------------- | ------------ |
| 7.11 | 到达 Christchurch，取车补给                                                    | Christchurch |
| 7.12 | Christchurch → Geraldine / Fairlie → Lake Tekapo → Lake Pukaki → Twizel | Twizel       |
| 7.13 | Twizel → Lake Pukaki → Aoraki / Mt Cook → Wanaka                        | Wanaka       |
| 7.14 | Wanaka 早出发 → Queenstown → 下午跳伞                                    | Queenstown   |
| 7.15 | Milford Sound 一日巴士团，Queenstown 出发返回                                     | Queenstown   |
| 7.16 | Queenstown → Arrowtown → Cromwell → Lindis Pass → Omarama → Twizel      | Twizel       |
| 7.17 | Twizel → Lake Tekapo / Fairlie / Geraldine → Christchurch → 返程          | 不住宿          |

## 每日行程

### 第 1 天｜7.11｜到达 Christchurch

**路线**

**Christchurch Airport → 取车 → 超市补给 → 酒店**

这天只做准备，不建议继续往 Tekapo / Twizel 开。冬季夜路、陌生右舵/左侧通行、乡村道路照明少，不值得冒险。

**路程时间**

| 路段                                     |         时间 |
| -------------------------------------- | ---------: |
| Christchurch Airport → CBD / Riccarton | 约 20–30 分钟 |
| CBD / Riccarton → 超市 / 酒店              | 约 10–20 分钟 |

**建议安排**

下午或晚上到达后：

1. 取车
2. 检查保险、雪链、轮胎、雨刷、玻璃水
3. 去超市补给
4. 入住休息

**补给建议**

可以买：

* 水、早餐、零食、热饮
* 简单午餐，例如三明治、面包、罐头、即食饭
* 湿巾、纸巾、垃圾袋
* 手套、帽子、暖宝宝
* 手机支架、车充、离线地图

**住宿地点**

推荐三个区域：

| 区域             | 适合情况           |
| -------------- | -------------- |
| **Airport 附近** | 航班晚、第二天想直接出城   |
| **Riccarton**  | 吃饭、超市方便，停车相对容易 |
| **CBD**        | 想简单逛市区、吃饭方便    |

更推荐 **Riccarton 或 Airport 附近**，因为第二天要往 Twizel 方向开，停车和出城更方便。

### 第 2 天｜7.12｜Christchurch → Lake Tekapo → Lake Pukaki → Twizel

**路线**

**Christchurch → Geraldine → Fairlie → Lake Tekapo → Lake Pukaki → Twizel**

这是从 Christchurch 进入 Mackenzie Basin 的经典路线。Twizel 是去 Mt Cook 和 Mackenzie 区域很方便的基地，新西兰官方旅游网站也把 Twizel 描述为探索 Mt Cook National Park 和 Mackenzie Basin 的实用基地。([新西兰官网][1])

**路程时间**

| 路段                                                    |         预计驾驶时间 |
| ----------------------------------------------------- | -------------: |
| Christchurch → Geraldine                              |         约 2 小时 |
| Geraldine → Fairlie                                   |        约 30 分钟 |
| Fairlie → Lake Tekapo                                 |     约 30–40 分钟 |
| Lake Tekapo → Lake Pukaki Viewpoint / Peter’s Lookout |     约 35–45 分钟 |
| Lake Pukaki → Twizel                                  |     约 15–25 分钟 |
| **当天纯驾驶合计**                                           | **约 4–4.5 小时** |

Christchurch 到 Queenstown 全程约 480km、不停车约 6 小时；把它拆成 Christchurch → Twizel → Wanaka → Queenstown，比一天硬开舒服很多。([PETRINA DARRAH][2])

**建议时间表**

| 时间          | 安排                            |
| ----------- | ----------------------------- |
| 08:30–09:00 | Christchurch 出发               |
| 10:45–11:15 | Geraldine 休息、上厕所、咖啡           |
| 11:45–12:15 | Fairlie，可买午餐或补给               |
| 12:45–14:00 | Lake Tekapo                   |
| 14:30–15:30 | Lake Pukaki / Peter’s Lookout |
| 16:00 前后    | 到 Twizel 入住                   |
| 晚上          | Twizel 吃饭、休息；天气好可看星空          |

**景点安排**

**Lake Tekapo**

建议停留 **1–1.5 小时**。

可以看：

* **Church of the Good Shepherd**
* Lake Tekapo 湖边
* 镇中心简单午餐 / 咖啡

冬季如果天气好，Tekapo 很漂亮；但这天不要停太久，因为还要去 Pukaki 和 Twizel。

**Lake Pukaki**

建议停留 **30–60 分钟**。

重点看：

* **Lake Pukaki Viewpoint**
* **Peter’s Lookout**
* 远眺 Aoraki / Mt Cook

这段是 7 天里最值得拍照的湖景之一，天气好时 Lake Pukaki + Mt Cook 非常出片。

**住宿地点**

**住 Twizel**

推荐住：

* Twizel 镇中心附近
* Lake Ruataniwha 附近
* 带停车位的 motel / apartment

Twizel 比 Tekapo 通常更安静，也更接近第二天的 Mt Cook 路线。

### 第 3 天｜7.13｜Twizel → Lake Pukaki → Aoraki / Mt Cook → Wanaka

这是全程最重要的一天之一。

**路线**

**Twizel → Lake Pukaki → Aoraki / Mt Cook Village → Hooker Valley / Tasman Glacier View → Omarama → Lindis Pass → Wanaka**

**路程时间**

| 路段                                     |         预计驾驶时间 |
| -------------------------------------- | -------------: |
| Twizel → Lake Pukaki                   |     约 15–20 分钟 |
| Lake Pukaki → Aoraki / Mt Cook Village |     约 45–60 分钟 |
| Mt Cook Village → Omarama              |       约 1.5 小时 |
| Omarama → Lindis Pass → Wanaka         |   约 1小时45分–2小时 |
| **当天纯驾驶合计**                            | **约 4–4.5 小时** |

**建议时间表：天气好版**

| 时间          | 安排                                       |
| ----------- | ---------------------------------------- |
| 08:00       | Twizel 出发                                |
| 08:20–08:50 | Lake Pukaki / Peter’s Lookout 简单拍照       |
| 09:30       | 到 Aoraki / Mt Cook Village               |
| 09:45–13:00 | Hooker Valley Track                      |
| 13:00–14:00 | 午餐 / 热饮 / 休息                             |
| 14:00–17:30 | Mt Cook → Omarama → Lindis Pass → Wanaka |
| 17:30–18:00 | 到 Wanaka 入住                              |

Hooker Valley Track 全程约 10km，官方旅游信息给出的往返时间通常是 **3–4 小时**。([新西兰官网][3]) DOC 的 Aoraki/Mt Cook 步道资料也标注 Hooker Valley Track 约 **3 小时往返**，如果从村里出发还要额外加时间。([新西兰政府文件][4])

**天气差 / 路滑版**

如果天气差、风很大、下雪、路面结冰，建议不要硬走全程 Hooker Valley。改成：

| 景点                                  |           停留 |
| ----------------------------------- | -----------: |
| Lake Pukaki / Peter’s Lookout       |     30–45 分钟 |
| Mt Cook Village                     |        30 分钟 |
| Tasman Glacier View / Blue Lakes 短线 |     45–90 分钟 |
| Kea Point Track 部分短走                | 视路况 30–90 分钟 |

这样可以保留 Mt Cook 体验，又不会把下午去 Wanaka 的时间压得太晚。

**午餐建议**

Mt Cook Village 附近选择有限，建议前一天或早上在 Twizel 准备：

* 三明治
* 面包 / 饼干 / 能量棒
* 热水 / 热饮
* 简单水果

**住宿地点**

**住 Wanaka**

推荐区域：

| 区域                       | 适合情况         |
| ------------------------ | ------------ |
| **Wanaka Town Centre**   | 吃饭方便，晚上可湖边散步 |
| **Lakefront 附近**         | 景色好，但可能贵     |
| **镇外 motel / apartment** | 停车方便，性价比高    |

### 第 4 天｜7.14｜Wanaka → Queenstown｜下午跳伞

这天上午从湖区进入 Queenstown，下午安排跳伞。跳伞受天气影响很大，建议预约 **14:00–15:00 左右的场次**，并把午餐、入住和市区活动都做成可调整项。

**路线选择**

**方案 A：天气好，走 Crown Range**

**Wanaka → Cardrona → Crown Range → Arrow Junction → Queenstown**

优点是风景好，路程短。
缺点是冬季有冰雪风险，山路驾驶压力更大。

**方案 B：天气差，走 Cromwell**

**Wanaka → Cromwell → Kawarau Gorge → Queenstown**

优点是更稳，冬季更保守。
缺点是稍微绕一点。

建议默认看当天路况：
**路况好走 Crown Range；下雪、结冰、雾大就走 Cromwell。**

**路程时间**

| 路线                                |   预计驾驶时间 |
| --------------------------------- | -------: |
| Wanaka → Queenstown，经 Crown Range | 约 1.5 小时 |
| Wanaka → Queenstown，经 Cromwell    |   约 2 小时 |

**建议时间表**

| 时间          | 安排                                      |
| ----------- | --------------------------------------- |
| 08:30–09:30 | Wanaka 湖边、That Wanaka Tree，简单早餐 / 咖啡       |
| 09:30–11:30 | 开往 Queenstown，视路况选择 Crown Range 或 Cromwell |
| 11:30–12:30 | 抵达 Queenstown，午餐 / 停车 / 简单补给              |
| 12:30–13:30 | 入住或先寄存行李，确认跳伞集合地点和接送时间                  |
| 13:30–17:00 | Queenstown 跳伞，含集合、装备、飞行、跳伞和返回             |
| 17:00–18:30 | Lake Wakatipu 湖边轻松散步 / Queenstown Gardens |
| 晚上          | Queenstown 晚餐，早点休息，为 7.15 早出发留体力          |

**跳伞安排**

建议把 Queenstown 跳伞作为这天的核心活动，不再塞太多沿途景点。

* 预约下午场，优先选择 **14:00–15:00** 左右的起跳 / 集合时间。
* 早上尽量 09:30 前从 Wanaka 出发，给山路、停车、午餐和入住留缓冲。
* 跳伞当天不要把贵重或易碎物品放在车内显眼位置，随身只带必要证件、手机和保暖外套。
* 如果天气取消，现场通常会改期；但 7.15 已安排 Milford Sound 一日团，改期空间有限，所以 7.14 下午要尽量预留完整。

**可停景点**

如果走 Crown Range：

* Cardrona Hotel 外观短停
* Crown Range Lookout，路况好再停

如果走 Cromwell：

* Cromwell 镇
* Kawarau Gorge
* AJ Hackett Bungy Bridge 外观短停，不参加项目也可以看一眼

这天因为下午有跳伞，沿途停留建议控制在 **10–15 分钟短停**，不要安排正式游玩。

**住宿地点**

**住 Queenstown 两晚：7.14、7.15**

因为 7.15 要 Milford Sound 巴士团，建议优先考虑：

| 区域                             | 评价                      |
| ------------------------------ | ----------------------- |
| **Queenstown CBD / Lakefront** | 最方便，很多巴士团接送点在附近         |
| **Frankton**                   | 停车、超市方便，住宿可能便宜，但确认巴士团接送 |
| **Arthurs Point**              | 安静，但对 Milford 团接送不一定方便  |

优先选 **可以被 Milford Sound 巴士团 pickup 的住宿**，否则清晨要自己走去集合点会比较麻烦。

### 第 5 天｜7.15｜Milford Sound 一日巴士团｜住 Queenstown

这天是全程最累的一天，但选择巴士团是正确的。Queenstown 到 Milford Sound 单程很长，RealNZ 明确说明这是一个长日行程，单程约 **5 小时**，有高山驾驶和多变天气，所以很多游客会选择 coach-and-cruise。([realnz.com][5])

**典型路线**

**Queenstown → Te Anau → Eglinton Valley → Mirror Lakes → Homer Tunnel → Milford Sound → 游船 → Queenstown**

**时间安排**

不同公司时间略有差异，但大概是：

| 时间          | 安排                  |
| ----------- | ------------------- |
| 06:30–07:30 | Queenstown 接送 / 集合  |
| 09:30–10:30 | Te Anau 休息          |
| 10:30–13:00 | Milford Road 沿途风景停留 |
| 13:00–15:30 | Milford Sound 游船    |
| 15:30–20:00 | 返回 Queenstown       |

Tripadvisor 上的 Queenstown 出发 Milford Sound day tour 也常见标注约 **13 小时**。([Tripadvisor][6])

**建议带什么**

* 防水外套
* 保暖层
* 手套、帽子
* 充电宝
* 简单早餐和零食
* 晕车药可以提前准备，但按药品说明使用
* 相机 / 手机备用电量

**住宿地点**

**继续住 Queenstown**

这晚别安排其他活动，第二天还要开回 Twizel。

### 第 6 天｜7.16｜Queenstown → Arrowtown → Cromwell → Lindis Pass → Omarama → Twizel

这天是返程缓冲日，安排得很好。因为 7.15 很累，所以 7.16 不要塞长徒步。

**路线**

**Queenstown → Arrowtown → Cromwell → Lindis Pass → Omarama → Twizel**

**路程时间**

| 路段                     |         预计驾驶时间 |
| ---------------------- | -------------: |
| Queenstown → Arrowtown |        约 25 分钟 |
| Arrowtown → Cromwell   |        约 45 分钟 |
| Cromwell → Lindis Pass |         约 1 小时 |
| Lindis Pass → Omarama  |        约 40 分钟 |
| Omarama → Twizel       |     约 25–30 分钟 |
| **当天纯驾驶合计**            | **约 3.5–4 小时** |

Queenstown 到 Lake Tekapo 经 Cromwell 约 3 小时，继续分到 Twizel / Omarama 一线非常顺路。([My Queenstown Diary][7])

**建议时间表**

| 时间          | 安排                              |
| ----------- | ------------------------------- |
| 09:30       | Queenstown 出发                   |
| 10:00–11:30 | Arrowtown 老街、咖啡                 |
| 12:15–13:30 | Cromwell / Old Cromwell Town 午餐 |
| 14:30       | Lindis Pass Viewpoint 短停        |
| 15:30       | Omarama 休息、加油、厕所                |
| 16:00–16:30 | 到 Twizel 入住                     |

**景点细化**

**Arrowtown**

停留 **1–1.5 小时**。

适合：

* 老街
* 河边短走
* 咖啡 / 早餐
* 拍冬季小镇氛围

**Cromwell**

停留 **1 小时左右**。

适合：

* Old Cromwell Town
* Lake Dunstan 湖边
* 午餐 / 咖啡 / 加油

**Lindis Pass**

停留 **10–20 分钟**。

这里只建议短停拍照。冬季可能风大、路边结冰，不建议久留或乱走到路外。

**Omarama**

停留 **15–30 分钟**。

适合休息、上厕所、加油、买热饮。
这天不要再安排额外长项目，早点到 Twizel 比较稳。

**住宿地点**

**住 Twizel**

选择带停车位、厨房或微波炉的住宿会更舒服。7.17 要早起返程，所以住宿位置不需要追求景观，方便停车和出发更重要。

### 第 7 天｜7.17｜Twizel → Christchurch → 返程

**路线**

**Twizel → Lake Tekapo → Fairlie → Geraldine → Christchurch Airport**

**路程时间**

| 路段                               |           预计驾驶时间 |
| -------------------------------- | ---------------: |
| Twizel → Lake Tekapo             |       约 45–60 分钟 |
| Lake Tekapo → Fairlie            |       约 30–40 分钟 |
| Fairlie → Geraldine              |          约 30 分钟 |
| Geraldine → Christchurch Airport |       约 2–2.5 小时 |
| **当天纯驾驶合计**                      | **约 3.5–4.5 小时** |

Twizel 到 Christchurch 约 284km，正常路况驾驶约 3小时46分左右。([Wanderlog][8])

**下午 / 晚上航班版本**

如果航班是 **16:00 以后**，可以这样走：

| 时间          | 安排                        |
| ----------- | ------------------------- |
| 08:00       | Twizel 出发                 |
| 09:00       | Lake Tekapo 简单咖啡 / 拍照     |
| 10:30       | Fairlie / Geraldine 休息    |
| 13:00–13:30 | 到 Christchurch Airport 附近 |
| 13:30–14:30 | 还车                        |
| 之后          | 值机、返程                     |

## 关键提醒

**7.13 不要太晚离开 Mt Cook。**
即使 Hooker Valley 天气好，下午 2 点左右也应该准备离开去 Wanaka，不然冬季天黑后开 Lindis Pass 会更累。

**7.14 Wanaka 到 Queenstown 不强求 Crown Range。**
天气好走 Crown Range；路面有冰雪就走 Cromwell。

**7.15 Milford Sound 后不要安排夜间活动。**
一日团很长，第二天还要开回 Twizel。

**7.17 航班最好在下午 4 点以后。**
这样 Twizel → Christchurch 返程比较稳。如果航班早，7.16 要改住 Christchurch。
