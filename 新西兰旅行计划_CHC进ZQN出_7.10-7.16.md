## 行程原则

* 7 月是新西兰南岛冬季，自驾尽量安排在白天，避免夜间长距离驾驶。
* 每天出发前检查天气、路况和租车公司的雪链要求，Crown Range、Lindis Pass、雪场道路尤其需要谨慎。
* Aoraki / Mt Cook 和 Milford Sound 的行程都高度依赖天气，优先保证安全和缓冲时间。
* 这版行程不再从 Queenstown 折返 Christchurch，整体更顺路，也更适合第一次南岛自驾。
* Brisbane 只有一个白天，不建议安排远郊或强度太高的项目。更有意思的优先项是 Lone Pine + Mt Coot-tha 或 Story Bridge Climb；轻松备选是 South Bank、CBD、QAGOMA / Queensland Museum。

## 天气与冬季驾驶调整

根据 7 月 7 日查询到的预报，后半程低温和雨雪风险比较明显，行程按冬季保守节奏执行。

| 日期 | 主要地点 | 天气 / 温度重点 | 对行程的影响 |
| ---- | -------- | --------------- | ------------ |
| 7.10 | Christchurch | 约 9°C / 1°C | 到达日无长途驾驶，安排不变 |
| 7.11 | Christchurch → Lake Tekapo | Tekapo 约 4°C / -9°C | 傍晚和夜间很冷，尽量 15:00 前到 Tekapo |
| 7.12 | Tekapo / Mt Cook / Wanaka | Tekapo 约 3°C / -6°C，Mt Cook 早上零下，Wanaka 约 8°C / -1°C | 暗冰风险高，Mt Cook 徒步减少为 Kea Point + Tasman，Hooker 开放段改为可选 |
| 7.13 | Wanaka → Queenstown | Wanaka / Queenstown 有雨夹雪或雨，Queenstown 约 5°C / -4°C | 默认走 Cromwell / Kawarau Gorge，不走 Crown Range |
| 7.14 | Queenstown | 约 7°C / -3°C，下午可能有小雨 | 跳伞和滑雪都可能受天气影响，保留 Glenorchy / 市区备选 |
| 7.15 | Queenstown / Milford Sound | Queenstown 有雪转雨，Milford Sound 有雪 / 雨，道路降雪警告 | 巴士团可能延误或调整，务必按确认单提前上车 |
| 7.16 | Queenstown → Brisbane | Queenstown 约 3°C / -6°C，有雨雪 | 上午只做轻松活动，去机场和还车都提前 |
| 7.17 | Brisbane | 约 21°C / 10°C，多晴 | Brisbane 一日游正常执行 |

冬季驾驶统一调整：

* 所有南岛跨城驾驶时间在原纯驾驶基础上额外预留 **30–60 分钟**。
* 7.12 Tekapo → Mt Cook → Wanaka 额外预留 **60–90 分钟**，并减少徒步项目。
* 7.13 Wanaka → Queenstown 默认走 **Cromwell / Kawarau Gorge**，Crown Range 只在当天路况明确良好、允许通行且雪链要求可满足时考虑。
* 每天早上出发前检查 NZTA Journey Planner、QLDC Winter Road Reports、MetService road snowfall warnings 和租车公司雪链要求。
* 暗冰常见于清晨、阴影路段、桥面、湖边和山口。低温天不要急刹、急转或贴车太近。

## Day 3｜7.12 Lake Tekapo → Aoraki / Mt Cook → Wanaka

**住宿：Wanaka**

推荐路线：

**Lake Tekapo → Lake Pukaki → Aoraki / Mt Cook Village → Hooker Valley 开放段 / Kea Point / Tasman Glacier View → Omarama → Lindis Pass → Wanaka**

预计驾驶时间：

| 路段                             |            时间 |
| ------------------------------ | ------------: |
| Lake Tekapo → Lake Pukaki      |     约 35–45分钟 |
| Lake Pukaki → Mt Cook Village  |     约 45–60分钟 |
| Mt Cook Village → Omarama      |       约 1.5小时 |
| Omarama → Lindis Pass → Wanaka |    约 1.75–2小时 |
| **合计纯驾驶**                      | **约 4.5–5小时** |
| **冬季缓冲后**                      | **约 5.5–6.5小时** |

低温优化版本：

| 时间          | 安排                            |
| ----------- | ----------------------------- |
| 08:15       | Lake Tekapo 出发，避开最冷清晨路面     |
| 09:00–09:30 | Lake Pukaki / Peter’s Lookout |
| 10:15       | 到 White Horse Hill car park   |
| 10:30–11:45 | Kea Point Track                |
| 12:00–12:45 | Mt Cook Village 午餐 / 咖啡        |
| 13:00–13:45 | Tasman Glacier View Track      |
| 13:45–14:15 | 机动：只在路况和体力都好时短走 Hooker 开放段 |
| 14:15–18:15 | 开往 Wanaka，途中按需在 Omarama 休息    |
| 18:15–18:45 | 到 Wanaka                      |

说明：

* Hooker Valley Track 目前只开放到 Kakīroa / Mount Sefton View Lookout，约 4.3 km 往返、1 小时左右，不是完整 Hooker Valley。
* 这段仍然值得走，可以看 Mueller Lake、Mueller Glacier、Mount Sefton 一带，但对 Aoraki / Mt Cook 主峰视野有限。
* Kea Point Track 是目前更适合作为完整 Hooker Valley 替代的短线，从 White Horse Hill 出发约 1 小时往返。
* Tasman Glacier View Track 很短，约 30–45 分钟，适合作为离开 Mt Cook 前的补充。
* 因为前一晚住 Lake Tekapo，比住 Twizel 多约 40–60 分钟车程；同时早上低温明显，这天改为 8:15 左右出发，并用减少徒步来换驾驶安全。
* 默认保留 **Kea Point + Tasman Glacier View**。Hooker Valley Track 开放段改为可选，不再默认三条都走。

天气差版本：

如果风大、下雪、云很低或路面滑，不建议硬走三条步道。可以按优先级删减：

1. 优先保留 Kea Point Track
2. 其次保留 Tasman Glacier View Track
3. 最后再考虑 Hooker Valley Track 开放段

天气差版本可以改成：

* Lake Pukaki 拍照
* Mt Cook Village
* Kea Point Track 短走
* Tasman Glacier View / Blue Lakes 短停
* 提前去 Wanaka

驾驶提醒：

* Lake Tekapo → Lake Pukaki、Pukaki → Mt Cook、Lindis Pass 一线都可能有暗冰，尤其是清晨、湖边、桥面和阴影路段。
* 如果 NZTA / MetService 对 Lindis Pass 或 Mt Cook 区域有雪冰警告，直接删掉 Hooker 开放段，必要时只在 Lake Pukaki 和 Mt Cook Village 短停。
* 这天不要追求到 Wanaka 前再加额外景点，目标是天黑前安全抵达。

主要景点：

* Lake Pukaki
* Aoraki / Mt Cook Village
* Hooker Valley Track 开放段
* Kea Point Track
* Tasman Glacier View
* Lindis Pass
* Wanaka 湖边

## Day 4｜7.13 Wanaka → Queenstown

**住宿：Fernhill, Queenstown**

已订住宿位于 Fernhill。Fernhill 在 Queenstown 市区西侧山坡上，湖景好，但进出市区需要预留额外时间；晚餐、活动集合和停车都建议提前规划。

推荐路线有两个。按当前低温和雨夹雪预报，**默认走方案 B：Cromwell / Kawarau Gorge**。

### 方案 A｜天气好，走 Crown Range

**Wanaka → Cardrona → Crown Range → Queenstown**

优点是风景更好，但冬季山路风险更高。只有在天气、路况和雪链要求都明确可控时再走；如有结冰、雪链要求或能见度差，直接放弃。

### 方案 B｜天气差，走 Cromwell

**Wanaka → Cromwell → Kawarau Gorge → Queenstown**

更稳，冬季作为默认路线。

预计驾驶时间：

| 路线            |      时间 |
| ------------- | ------: |
| 经 Crown Range | 约 1.5小时 |
| 经 Cromwell    |   约 2小时 |
| 经 Cromwell 冬季缓冲后 | 约 2.5–3小时 |

建议时间：

| 时间          | 安排                                    |
| ----------- | ------------------------------------- |
| 09:30–11:00 | Wanaka 湖边、That Wanaka Tree            |
| 11:00–12:00 | 可选 Mount Iron 短走 / Eely Point         |
| 12:00–13:00 | 午餐                                    |
| 13:00–15:30 | 经 Cromwell / Kawarau Gorge 开往 Queenstown |
| 15:30–16:30 | 前往 Fernhill 住宿，入住 / 停车 / 放行李          |
| 16:30–18:00 | Lake Wakatipu、Queenstown Gardens、市区短走 |
| 晚上          | Queenstown 市区晚餐，或回 Fernhill 附近休息       |

主要景点：

* Lake Wanaka
* That Wanaka Tree
* Cardrona Hotel，若走 Crown Range
* Kawarau Gorge，若走 Cromwell
* Lake Wakatipu
* Queenstown Gardens

说明：

* Fernhill 到 Queenstown CBD 车程通常不长，但旺季、饭点和停车会拉长时间。
* 如果晚上去市区吃饭，建议提前查停车，或使用出租车 / rideshare，避免夜间找车位和山路停车压力。
* 7.13 Wanaka / Queenstown 预报有雨夹雪或雨，Crown Range 不作为默认路线。

## Day 6｜7.15 Milford Sound 一日巴士团

**住宿：Fernhill, Queenstown**

这天不自驾，报 Queenstown 出发的 Milford Sound 巴士团。

7.15 Milford Sound / Milford Road 有雪雨和道路降雪风险。选择巴士团是正确的，不建议改为自驾；当天可能出现延误、改线或取消，以供应商通知为准。

上车安排：

* 供应商不提供指定范围外酒店接送；Fernhill 住宿不在直接接送名单内。
* 预订时如果酒店不在列表中，按供应商要求选择 **Millennium Hotel Queenstown** 作为接送参考点。
* 实际上车点不是酒店大堂，而是电子确认单中指定的酒店附近公共巴士站；部分地点需要步行 3–5 分钟。
* 07:10 为上车时间，至少提前 15 分钟到达指定上车点，建议 **06:50–06:55 到**。
* 司机通常不会提前打电话；错过接送将无法参加行程且不退款。

典型时间：

| 时间          | 安排                                      |
| ----------- | --------------------------------------- |

| 07:10       | Milford Sound 巴士团上车                       |

| 13:30–15:30 | Milford Sound 游船                          |

注意：

* 这天非常长，前一晚早点睡。
* 回来后不要再安排夜间活动。
* 第二天返程，建议提前整理好行李。
* 前一晚再次核对电子确认单里的上车点、步行路线和到达时间。
* 不建议自己开车去集合点后长时间停车；更稳的是打车 / rideshare 到上车点。

## Day 8｜7.17 Brisbane 一日游 → Sydney

**住宿：返程**

这天适合安排 Brisbane 半日游。Brisbane Airport 官方建议国内航班提前 1 小时到机场；考虑到行李、周五晚高峰和不熟悉机场，建议 **16:15–16:30 到达机场**。

如果想要更有体验感，优先考虑 **Lone Pine + Mt Coot-tha** 或 **Story Bridge Climb + Howard Smith Wharves**；如果天气差或想轻松，再选市区 / 博物馆方案。
