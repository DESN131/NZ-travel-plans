# 新西兰南岛 7 天自驾计划：7.10–7.17

| 日期       | 住宿           | 核心行程                                                               |
| -------- | ------------ | ------------------------------------------------------------------ |
| **7.10** | Christchurch | 到达 Christchurch，取车、超市补给                                            |
| **7.11** | Twizel       | Christchurch → Tekapo → Lake Pukaki → Twizel                       |
| **7.12** | Wanaka       | Twizel → Lake Pukaki → Aoraki / Mt Cook → Wanaka                   |
| **7.13** | Queenstown   | Wanaka → Queenstown                                                |
| **7.14** | Queenstown   | Queenstown 一日游 / 滑雪                                                |
| **7.15** | Queenstown   | Milford Sound 一日巴士团                                                |
| **7.16** | Twizel       | Queenstown → Arrowtown / Cromwell → Lindis Pass → Omarama → Twizel |
| **7.17** | 返程           | Twizel → Christchurch → 返程                                         |

---

## 7.10 到达 Christchurch

**住宿：Christchurch**

建议住：

* **Christchurch Airport 附近**：如果到达较晚，最省事
* **Riccarton / CBD**：吃饭、超市补给更方便

当天安排：

**到达 → 取车 → 超市补给 → 休息**

建议补给：水、零食、早餐、车载充电线、保暖衣物、手套、防水外套。
这天不要赶夜路去 Tekapo 或 Twizel。

---

## 7.11 Christchurch → Tekapo → Lake Pukaki → Twizel

**住宿：Twizel**

推荐路线：

**Christchurch → Geraldine → Fairlie → Lake Tekapo → Lake Pukaki → Twizel**

预计驾驶时间：

| 路段                        |            时间 |
| ------------------------- | ------------: |
| Christchurch → Geraldine  |         约 2小时 |
| Geraldine → Fairlie       |        约 30分钟 |
| Fairlie → Lake Tekapo     |     约 30–40分钟 |
| Lake Tekapo → Lake Pukaki |     约 35–45分钟 |
| Lake Pukaki → Twizel      |        约 20分钟 |
| **合计纯驾驶**                 | **约 4–4.5小时** |

建议时间：

| 时间          | 安排                     |
| ----------- | ---------------------- |
| 09:00       | Christchurch 出发        |
| 11:00       | Geraldine / Fairlie 休息 |
| 12:00–13:30 | Lake Tekapo            |
| 14:00–15:00 | Lake Pukaki            |
| 15:30–16:00 | 到 Twizel               |
| 晚上          | Twizel 休息，天气好可看星空      |

主要景点：

* Lake Tekapo
* Church of the Good Shepherd
* Lake Pukaki Viewpoint
* Peter’s Lookout
* Twizel 小镇补给、休息

## 7.12 Twizel → Lake Pukaki → Aoraki / Mt Cook → Wanaka

**住宿：Wanaka**

推荐路线：

**Twizel → Lake Pukaki → Aoraki / Mt Cook Village → Hooker Valley / Tasman Glacier View → Omarama → Lindis Pass → Wanaka**

预计驾驶时间：

| 路段                             |            时间 |
| ------------------------------ | ------------: |
| Twizel → Lake Pukaki           |     约 15–20分钟 |
| Lake Pukaki → Mt Cook Village  |     约 45–60分钟 |
| Mt Cook Village → Omarama      |       约 1.5小时 |
| Omarama → Lindis Pass → Wanaka |    约 1.75–2小时 |
| **合计纯驾驶**                      | **约 4–4.5小时** |

天气好版本：

| 时间          | 安排                            |
| ----------- | ----------------------------- |
| 08:30       | Twizel 出发                     |
| 09:00–10:00 | Lake Pukaki / Peter’s Lookout |
| 10:30       | 到 Mt Cook Village             |
| 10:45–14:00 | Hooker Valley Track           |
| 14:00–15:00 | 午餐 / 休息                       |
| 15:00–17:30 | 开往 Wanaka                     |
| 17:30–18:00 | 到 Wanaka                      |

天气差版本：

如果风大、下雪、云很低或路面滑，不建议硬走完整 Hooker Valley。可以改成：

* Lake Pukaki 拍照
* Mt Cook Village
* Kea Point Track 短走
* Tasman Glacier View / Blue Lakes 短停
* 提前去 Wanaka

主要景点：

* Lake Pukaki
* Aoraki / Mt Cook Village
* Hooker Valley Track
* Kea Point Track
* Tasman Glacier View
* Lindis Pass
* Wanaka 湖边

## 7.13 Wanaka → Queenstown

**住宿：Queenstown**

推荐路线有两个：

### 方案 A：天气好，走 Crown Range

**Wanaka → Cardrona → Crown Range → Queenstown**

优点是风景更好，但冬季山路风险更高。

### 方案 B：天气差，走 Cromwell

**Wanaka → Cromwell → Kawarau Gorge → Queenstown**

更稳，冬季更推荐作为默认备选。

预计驾驶时间：

| 路线            |      时间 |
| ------------- | ------: |
| 经 Crown Range | 约 1.5小时 |
| 经 Cromwell    |   约 2小时 |

建议时间：

| 时间          | 安排                                    |
| ----------- | ------------------------------------- |
| 09:30–11:00 | Wanaka 湖边、That Wanaka Tree            |
| 11:00–12:00 | 可选 Mount Iron 短走 / Eely Point         |
| 12:00–13:00 | 午餐                                    |
| 13:00–15:00 | 开往 Queenstown                         |
| 15:00–16:00 | 入住                                    |
| 傍晚          | Lake Wakatipu、Queenstown Gardens、市区吃饭 |

主要景点：

* Lake Wanaka
* That Wanaka Tree
* Cardrona Hotel，若走 Crown Range
* Kawarau Gorge，若走 Cromwell
* Lake Wakatipu
* Queenstown Gardens

## 7.14 Queenstown 一日游 / 滑雪

**住宿：Queenstown**

这天二选一。

## 方案 A：Queenstown + Glenorchy 轻松一日游

路线：

**Queenstown → Bennetts Bluff Lookout → Glenorchy → Queenstown**

预计驾驶时间：

| 路段                     |        时间 |
| ---------------------- | --------: |
| Queenstown → Glenorchy |    约 45分钟 |
| 往返                     | 约 1.5–2小时 |

建议时间：

| 时间          | 安排                                   |
| ----------- | ------------------------------------ |
| 09:30       | Queenstown 出发                        |
| 10:00       | Bennetts Bluff Lookout               |
| 11:00–13:00 | Glenorchy 湖边、码头                      |
| 13:00–14:00 | 午餐 / 咖啡                              |
| 15:00       | 回 Queenstown                         |
| 16:00–18:00 | Skyline Gondola / Queenstown Gardens |
| 晚上          | Queenstown 市区吃饭                      |

## 方案 B：滑雪日

推荐：

* **Coronet Peak**：更近，更适合轻松体验或初学者
* **The Remarkables**：景色更壮观，但山路压力更大

建议：

| 时间          | 安排               |
| ----------- | ---------------- |
| 07:30–08:30 | 出发 / 坐雪场 shuttle |
| 09:00–15:00 | 滑雪               |
| 16:00–17:00 | 回 Queenstown     |
| 晚上          | 早点休息             |

如果是第一次滑雪或不熟悉雪地驾驶，建议坐 shuttle，不建议自己开上雪场。

## 7.17 Twizel → Christchurch → 返程

推荐路线：

**Twizel → Lake Tekapo → Fairlie → Geraldine → Christchurch Airport**

预计驾驶时间：

| 路段                               |            时间 |
| -------------------------------- | ------------: |
| Twizel → Lake Tekapo             |     约 45–60分钟 |
| Lake Tekapo → Fairlie            |     约 30–40分钟 |
| Fairlie → Geraldine              |        约 30分钟 |
| Geraldine → Christchurch Airport |     约 2–2.5小时 |
| **合计纯驾驶**                        | **约 4–4.5小时** |

如果返程航班是 **下午 3 点以后**：

| 时间          | 安排                        |
| ----------- | ------------------------- |
| 08:00       | Twizel 出发                 |
| 09:00       | Tekapo 简单停留 / 咖啡          |
| 10:30       | Fairlie / Geraldine 休息    |
| 13:00–13:30 | 到 Christchurch Airport 附近 |
| 之后          | 还车、值机、返程                  |

如果返程航班是 **中午或更早**：

不要从 Twizel 当天开回 Christchurch。
这种情况建议把 **7.16 晚上改住 Christchurch 或机场附近**。

# 最关键提醒

**7.12 是全程最重要的风景日**：Lake Pukaki + Mt Cook，尽量早出发。
**7.13 Wanaka → Queenstown**：天气差就别走 Crown Range，改走 Cromwell。
**7.15 Milford Sound**：巴士团很长，晚上别安排活动。
**7.17 返程**：航班最好在下午 3 点以后；否则 7.16 直接住 Christchurch 更稳。
